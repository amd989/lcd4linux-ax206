lcd4linux-ax206 Package Repository

Debian / Ubuntu / Mint (apt)

Quick Install

curl -fsSL https://amd989.github.io/lcd4linux-ax206/setup.sh | sudo bash
sudo apt install lcd4linux-ax206

Manual Setup

curl -fsSL https://amd989.github.io/lcd4linux-ax206/gpg.key | sudo gpg --dearmor -o /usr/share/keyrings/lcd4linux-ax206.gpg
echo "deb [signed-by=/usr/share/keyrings/lcd4linux-ax206.gpg] https://amd989.github.io/lcd4linux-ax206 stable main" | sudo tee /etc/apt/sources.list.d/lcd4linux-ax206.list
sudo apt update
sudo apt install lcd4linux-ax206

Rocky Linux / RHEL / Fedora (dnf)

Quick Install

curl -fsSL https://amd989.github.io/lcd4linux-ax206/setup-rpm.sh | sudo bash
sudo dnf install lcd4linux-ax206

Manual Setup

sudo rpm --import https://amd989.github.io/lcd4linux-ax206/gpg.key
sudo tee /etc/yum.repos.d/lcd4linux-ax206.repo <<EOF
[lcd4linux-ax206]
name=lcd4linux-ax206
baseurl=https://amd989.github.io/lcd4linux-ax206/rpm/
enabled=1
gpgcheck=1
gpgkey=https://amd989.github.io/lcd4linux-ax206/gpg.key
EOF
sudo dnf install lcd4linux-ax206
